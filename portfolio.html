VIRTUAL PET SHELTER CONSOLE APPLICATION

Created a console application which mimics a pet shelter with employees, volunteers, and adoptable pets. Different employee types will see a different menu of possible actions to choose from. The employees and volunteers can each perform tasks such as feeding and playing with the animals. The manager has the capability to allow pets to be adopted.
